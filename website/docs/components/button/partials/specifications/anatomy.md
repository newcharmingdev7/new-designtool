## Anatomy

![Button anatomy](/assets/components/button/button-anatomy.png =643x*)

| Element          | Usage                                           |
|------------------|-------------------------------------------------|
| Text*            | Optional                                        |
| Container        | Required                                        |
| Leading Icon*    | Optional                                        |
| Focus Ring       | Focus state only                                |
| Trailing Icon*   | Optional                                        |

\* One of these must be present.
